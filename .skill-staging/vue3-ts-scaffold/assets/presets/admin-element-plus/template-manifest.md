# Admin Element Plus Template Manifest

This preset is a runnable admin-oriented Vue 3 template snapshot.

## Included capabilities

- Vite + TypeScript + Vue 3
- Element Plus with Chinese locale
- Auto-import for Vue APIs, Element Plus components, and Element Plus icons
- Pinia store setup
- Vue Router with auth and permission checks
- Axios request layer
- Login, forbidden, and not-found pages
- Example list module with composable-driven loading, empty, and error states

## Notes

- The auth and example APIs are mock-backed so the scaffold works without a live backend.
- `src/utils/request.ts` is still included for real backend integration.
- Replace the mock API layer once the backend contract is ready.
